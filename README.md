## Commit 规范

```
type(project): 描述
```

**type**: `feat` `fix` `refactor` `docs` `chore`

**示例**:
- `feat(Anlo): 添加元素选择器优化`
- `fix(LevitateBoat): 修复记忆索引问题`
- `chore(Madoka): 更新依赖`
