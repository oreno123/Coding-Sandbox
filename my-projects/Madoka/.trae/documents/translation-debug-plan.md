# 翻译功能丢失问题排查计划

## 问题描述
- 翻译功能之前可用，现在突然丢失
- 未修改过代码
- 需要定位问题根因

## 排查步骤

### 1. 检查代码完整性
- 验证 translation-popup.ts 是否存在且内容完整
- 验证 content/index.ts 中的翻译相关代码
- 验证 background/index.ts 中的翻译API代码

### 2. 检查构建输出
- 检查 dist 目录是否包含翻译相关代码
- 验证 manifest.json 是否正确配置

### 3. 检查扩展状态
- 扩展是否需要重新加载
- 扩展权限是否完整

### 4. 常见原因检查
- 扩展上下文失效 (Extension context invalidated)
- 浏览器更新导致的问题
- 缓存问题

## 排查后行动
根据排查结果，提供修复方案或编写功能解析文档。
