# Checklist

## 隐藏式记忆标签

- [ ] XML 注释格式的记忆标签正确嵌入到 LLM 回复中
- [ ] 用户看不到记忆标签（在显示前被移除）
- [ ] 系统能正确解析 XML 注释提取记忆数据
- [ ] 解析失败时有适当的错误处理

## 基于规则的记忆判定

- [ ] 内容分析器能识别个人信息（年龄、职业等）
- [ ] 内容分析器能识别用户偏好（喜欢/不喜欢）
- [ ] 内容分析器能识别重要事实（学习、工作目标等）
- [ ] 评分算法合理，阈值设定恰当
- [ ] 重要内容自动保存，无需 LLM 判断
- [ ] 日常寒暄等不重要内容被正确过滤

## 记忆上下文格式优化

- [ ] 记忆上下文以自然语言列表形式呈现
- [ ] 格式清晰易读，减少 token 消耗
- [ ] LLM 能正确理解并使用记忆上下文

## 记忆使用检测

- [ ] 能检测 LLM 是否引用了记忆内容
- [ ] 关键词匹配机制有效
- [ ] 记忆使用情况被正确记录
- [ ] 根据使用情况能调整记忆权重

## 代码质量

- [ ] 无 TypeScript 编译错误
- [ ] 代码符合项目风格
- [ ] 有适当的错误处理和日志

## 功能测试

- [ ] 完整对话流程测试：发送消息 → LLM 回复（含隐藏标签）→ 解析标签 → 保存记忆
- [ ] 测试记忆是否正确携带到后续对话
- [ ] 测试 LLM 是否正确使用记忆上下文
