# Levitate Boat 功能特性

## ✨ 核心功能

### 1. 智能对话系统
- 基于 LLM 的智能问答
- 支持多种 LLM 提供商（Ollama、OpenAI、Claude）
- 上下文记忆功能
- 流式响应支持

### 2. 记忆管理
- 自动提取和存储对话记忆
- 基于向量的语义检索
- 记忆导出/导入功能
- 标签系统

### 3. 黑金色暗色主题 🆕
- 深黑色背景 (#0a0a0a)
- 奢华金色点缀 (#d4af37)
- 金色渐变效果
- 柔和的光晕和阴影
- 优雅的视觉体验

### 4. 侧边栏布局 🆕
- 可折叠的对话历史侧边栏
- 对话列表管理
- 快速切换对话
- 新建对话功能
- 平滑的展开/折叠动画

## 🎨 视觉设计

### 配色方案

#### 浅色主题
```
背景色：#ffffff (白色)
主文字：#1a1a1a (深灰黑)
次要文字：#4b5563 (灰色)
强调色：#3b82f6 (蓝色)
```

#### 暗色主题（黑金色）
```
背景色：#0a0a0a (深黑)
侧边栏：#000000 (纯黑)
主文字：#e8e6e3 (米白)
次要文字：#c9b896 (浅金)
强调色：#d4af37 (金色)
渐变：#d4af37 → #f4d03f (金色渐变)
```

- 用户消息气泡：金色渐变

- 滚动条：半透明金色渐变

#### 光晕效果
- 页面背景：顶部金色光晕

- 按钮悬停：增强金色光晕

#### 边框样式
