# 扩展图标

请在本目录放置以下 PNG 文件，Chrome 会用作扩展图标：

| 文件名   | 尺寸    | 用途                         |
|----------|---------|------------------------------|
| 16.png   | 16×16   | 工具栏小图标、上下文菜单     |
| 32.png   | 32×32   | 工具栏图标（高 DPI）         |
| 48.png   | 48×48   | 扩展管理页 (chrome://extensions) |
| 128.png  | 128×128 | 安装提示、应用商店展示       |

**要求**：正方形、PNG 格式（可透明）。

**方式一**：用本目录下的 `icon.svg` 在线上转为 PNG  
- 打开 [cloudconvert.com/svg-to-png](https://cloudconvert.com/svg-to-png) 等工具  
- 上传 `icon.svg`，分别导出 16、32、48、128 尺寸，命名为上表文件名  

**方式二**：自己设计 16×16、32×32、48×48、128×128 四张 PNG，放入本目录即可。

放好后执行 `npm run build`，在 Chrome 中重新加载扩展即可看到图标。
