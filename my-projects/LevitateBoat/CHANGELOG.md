# 更新日志

## 2026-02-02 - 黑金色主题与侧边栏布局

### 新增功能

#### 🎨 黑金色暗色主题
- 全新的黑金色配色方案，营造高端奢华的视觉体验
- 深黑色背景 (#0a0a0a) 搭配金色点缀 (#d4af37)
- 金色渐变效果应用于：
  - 用户消息气泡
  - 发送按钮
  - 主题切换开关
  - 标题文字
  - 活跃对话项
- 微妙的金色光晕和阴影效果
- 金色渐变滚动条

#### 📂 侧边栏布局
- 新增可折叠的侧边栏（宽度 260px，折叠后 50px）
- 对话历史列表展示
  - 显示对话标题和时间
  - 当前活跃对话高亮显示
  - 悬停效果和点击切换
- "新建对话"按钮
- 侧边栏折叠/展开动画
- 主内容区自适应宽度

#### 🎯 界面优化
- 弹窗宽度从 420px 扩展到 720px
- 更好的空间利用率
- 流畅的过渡动画
- 响应式布局设计

### 样式细节

#### 暗色主题特色
- 背景：径向渐变，顶部带有金色光晕
- 边框：半透明金色边框
- 文字：米白色主文字 + 金色辅助文字
- 按钮：金色渐变背景，悬停时提升效果
- 输入框：深色背景，金色聚焦边框
- 滚动条：金色渐变，半透明效果

#### 侧边栏特色
- 对话项：圆角卡片设计
- 活跃项：左侧金色边框 + 金色光晕
- 图标：表情符号作为对话图标
- 按钮：金色渐变"新建对话"按钮

### 技术实现
- 使用 CSS 变量实现主题切换
- Vue 3 Composition API
- 响应式数据管理
- 平滑的 CSS 过渡动画
- 渐变和阴影效果的精细调整

### 兼容性
- 保持浅色主题不变
- 所有功能向后兼容
