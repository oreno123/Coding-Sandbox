# Levitate Boat

基于**本地记忆**与 **LLM** 的浏览器扩展：在保护隐私的前提下，让对话拥有长期记忆与个性化回复，并支持你通过标签与画像**持续学习**、越用越懂你。

---

## 设计理念：限制即保护

本插件的功能与边界，既是一种**限制**，也是一种**保护**。

- **数据留在本地**：对话、记忆、用户画像均存储在浏览器本地（IndexedDB），不上传到任何第三方服务器。你拥有完整控制权。
- **按对话隔离记忆**：每个对话拥有独立记忆空间，删除对话会一并删除该对话下的记忆，避免信息混杂与误用。
- **可控的自动化**：记忆由 LLM 从对话中**自动提取**，但你可以通过「自定义标签」「回复风格」「自定义说明」引导提取与回复方式，在自动化与可控之间取得平衡。
- **用户画像仅服务于你**：画像（常问、常用标签、回复风格）只用于优化**你自己的**对话体验，不用于训练或分享。

我们希望你在**合理、知情**的前提下使用本插件，并借由它养成**持续整理与反思**的习惯，让工具为人服务，而不是人被工具裹挟。

---

## 核心功能

- **记忆增强对话**：根据当前问题检索相关记忆，结合 LLM 生成回答。
- **自动记忆提取**：从对话中抽取事实、偏好、事件并打标签保存；支持**自定义标签**，提取时优先考虑你填写的标签。
- **用户画像**：统计你的常问与常用标签，用于调整回复风格；支持「回复风格」「自定义说明」与**自定义标签**的编辑与展示。
- **对话与标签管理**：多对话、多标签页；可关闭标签页而不删记录；新建对话时可设话题与回复风格；删除对话会同步删除该对话下的记忆。
- **流式输出与 Markdown**：回复流式展示，并实时渲染 Markdown。
- **主题与布局**：浅色/暗色主题（黑金色暗色主题）；侧栏与主区独立滚动；导出/导入 JSON 备份。

## 合理使用与持续学习

### 合理使用

1. **配置 API**：在「设置」中填写你信任的 LLM API（如 DeepSeek、Ollama、OpenAI 等）。密钥与对话内容仅发往你配置的地址，本插件不收集数据。
2. **善用话题与风格**：新建对话时填写「话题」和选择「回复风格」，有助于记忆与回复更贴合该对话场景。
3. **自定义标签**：在设置页「用户画像」中填写你常用的标签（如「工作」「学习」「生活」）。提取记忆时，模型会优先参考这些标签，使记忆分类更符合你的习惯。
4. **定期导出**：重要记忆建议定期在设置页「导出记忆」备份，防止误删或重装扩展导致数据丢失。
5. **关闭与删除**：不需要的对话可先「关闭标签」隐藏；确认不再需要时再在侧栏删除，此时会同步删除该对话下的所有记忆。

### 持续学习

- **画像会随使用而丰富**：你的提问与提取出的记忆标签会统计进「近期常问」「常用标签」，并在设置页以标签云等形式展示，便于你了解自己的使用习惯。
- **自定义说明与风格**：通过「回复风格」和「自定义说明」告诉助手你的偏好，助手会在后续回答与提取记忆中参考这些信息，实现**越用越贴合你**的效果。
- **标签的桥梁作用**：你填写的「自定义标签」与自动统计的「常用标签」一起，既约束了记忆的归类方式，也帮助你在大量记忆中快速定位与回顾，形成可持续的**学习与整理闭环**。

我们鼓励在知情、可控的前提下，把本插件当作**长期使用的学习与记忆外脑**，而不是一次性工具。

## 技术栈与开发

- **技术栈**：Vue 3、TypeScript、IndexedDB（Dexie）、Ollama/OpenAI 兼容 API
- **开发**：
  ```bash
  npm install
  npm run dev    # 监听模式
  npm run build  # 生产构建
  ```
- 构建产物在 `dist/`。Chrome 扩展管理页加载该目录即可。

---

## 配置说明

在扩展「设置」页可配置：

- **API**：类型（Ollama / OpenAI / Claude / 自定义）、地址、Key、Chat 模型、Embedding 模型（可选）
- **用户画像**：回复风格（简洁/适中/详细）、自定义说明、**自定义标签**（可增删）；可查看近期常问、常用标签与标签云
- **数据**：导出记忆、导入记忆（JSON）

默认示例为 DeepSeek（自定义 API），实际 Key 与地址需自行填写。
