# 项目概述

## 项目名称
**keyusee - 画中画录屏工具**

## 项目标语
免费的 Focusee 平替工具，纯 AI 开发，欢迎提出宝贵建议！  
从“录下来”到“讲明白”，通过先录后剪、自动着重、画中画与 3D 变换，让教程、演示与讲解视频制作更高效。

## 项目简介
这是一个基于 Electron + React + TypeScript 的桌面录屏与剪辑工具，面向教程录制、产品演示、问题复现与远程协作讲解场景。

## 关键组件
- **主进程 (Main Process)**: `src/main`
- **预加载脚本 (Preload)**: `src/preload`
- **渲染进程 (Renderer)**: `src/renderer` (React 应用)

## 依赖与配置
- Electron
- React
- TypeScript
- Material-UI (根据用户指示，将专注于使用 Material-UI)

## 版本历史
- **剪辑功能增强 (2026-03-16)**:
  - **按键显示优化**: 优化了剪辑窗口中键盘操作的展示，将原始键值(keycode)转换为易读的具体按键名称（如 `按键A`、`按键Ctrl+Alt+Shift+A`），提升了键鼠轨迹回放的可读性。
  - **时间轴交互优化**: 点击时间轴可精确定位并自动暂停预览画面，支持空格键播放/暂停快捷键。
  - **配置自动保存**: 实现了基于项目ID的配置自动保存与恢复机制，下次打开同一项目自动加载上次编辑状态。
  - **配置导入导出**: 新增配置存储与导入功能，支持跨项目复用剪辑参数设置。

- **高级视频剪辑界面重构 (2026-03-15)**:
  - **全新布局设计**: 实现了上（左中右）下（时间轨道）的专业视频剪辑软件布局。
  - **左侧配置面板**: 支持基础设置（画面比例、底图、自定义光标等）及高级属性（边距、圆角、阴影等）调节。
  - **高性能预览渲染**: 使用 HTML5 Canvas 与 `requestAnimationFrame` 实现了录屏、摄像头、底图、光标与键盘提示的多层同步渲染引擎，预览帧率支持 30fps+。
  - **右侧元素编辑**: 实现选中不同视频元素（录屏、摄像头、光标等）时的动态参数编辑面板。
  - **主进程 IPC 扩展**: 修改 `get-input-log-parsed` 返回原始时间戳及坐标，为高精度渲染提供数据支持。

- **历史录屏能力增强 (2026-03-15)**:
  - **录屏历史配置文件**: 新增 `recording-history.json`，每次保存录屏后写入录制时间、视频时长、保存名称和绝对路径。
  - **主窗口历史选项卡**: 在主窗口新增“历史录屏”选项卡，支持按列表查看历史录制记录并刷新。
  - **历史详情窗口**: 点击历史项可打开新窗口，支持摄像头视频预览、录屏视频预览、键鼠操作记录可读化列表展示。
  - **主进程 IPC 扩展**: 新增历史记录读取、单条记录查询、键鼠日志解析与详情窗口打开能力。

- **UI/UX Enhancements (2026-03-14)**:
  - **无边框窗口改造**: 将保存窗口和设置窗口改为无边框样式，实现了自定义标题栏（包含拖拽区域和关闭按钮）及窗口阴影效果，保持与主窗口风格一致。
  - **组件封装**: 创建了 `CustomWindow` 组件，统一管理无边框窗口的布局、拖拽逻辑和样式。
  - **摄像头窗口优化**: 实现了摄像头预览窗口的位置记忆功能，并添加了屏幕边界检查，防止窗口超出可视区域。
  - **主窗口交互**: 优化了主窗口关闭逻辑，点击关闭按钮时隐藏窗口而非退出程序，仅通过托盘菜单退出。

- **Bug Fixes**:
  - **自动录屏修复**: 移除了程序启动时自动开始倒计时录屏的逻辑。
  - **窗口销毁错误**: 修复了主窗口隐藏后再显示时可能出现的 "Object has been destroyed" 错误。
  - **倒计时二次打开修复 (2026-03-14)**: 修复录屏倒计时窗口第二次打开时无法正常从 3 开始计时的问题。倒计时页在窗口重新获得焦点或重新可见时会重置计数，主进程改为显示并聚焦窗口而非强制 reload，避免状态竞态导致计时异常。

- **Initial Commit**: 初始化项目结构，配置 Git 仓库。
