# 项目详细说明（keyusee - 画中画录屏工具）

项目定位：免费的 Focusee 平替工具，采用先录后剪工作流，支持画中画讲解、自动着重与 3D 变换效果，用于高效率制作可发布的讲解视频。

## 高级视频剪辑界面技术实现 (HistoryDetail.tsx)

### 1. 架构设计
- **布局分区**：
  - 左侧：全局配置面板（使用 Material-UI `Tabs` 实现多选项卡，调节比例、底图、光标等）。
  - 中间：高性能 Canvas 预览区。
  - 右侧：选中元素动态编辑面板。
  - 下侧：时间轴与播放控制（使用 Slider 与占位 Box）。

### 2. 状态管理
所有编辑器状态集中在 `HistoryDetail` 的 `config` 状态中，采用受控组件模式（Controlled Components）双向绑定至左侧与右侧的调节滑块（Slider）、选择器（Select）和输入框（TextField）。

```typescript
interface EditorConfig {
  aspectPreset: string;
  cameraShape: 'circle' | 'rect';
  backgroundImage: string;
  cursorImage: string;
  cursorSize: number;
  screenMargin: number;
  cameraScale: number;
  shadowBlur: number;
  shadowOpacity: number;
  shadowOffsetX: number;
  shadowOffsetY: number;
  screenRadius: number;
  cameraRadius: number;
}
```

### 3. 高性能渲染引擎 (Canvas + requestAnimationFrame)
为保证 30fps+ 的多层视频流畅预览，摒弃了传统的 DOM 层叠（z-index）方案，直接使用 HTML5 `<canvas>` 进行帧级合并渲染：
- **虚拟分辨率**：内部采用固定 1920 宽度的虚拟坐标系，利用 `ctx.scale` 自适应不同显示器尺寸，确保抗锯齿。
- **渲染循环**：在 `playing` 状态下通过 `requestAnimationFrame` 驱动，逐层绘制：
  1. 底图 (`ctx.drawImage`) 
  2. 录屏视频 (带 `roundRect` 剪裁与阴影属性)
  3. 摄像头视频 (带 `arc` 或 `roundRect` 剪裁与发光边框)
  4. 鼠标指针 (根据当前 `video.currentTime` 匹配 `inputLog` 中的 `timestamp` 计算坐标)
  5. 键盘提示 (在画布底部居中渲染半透明气泡文字)

### 4. IPC 与日志解析增强
为主进程 `get-input-log-parsed` 接口补充了 `timestamp`, `x`, `y` 字段的返回。
这使得渲染进程可以将相对视频的播放时间 (`currentTime * 1000`) 与键鼠操作记录精准同步，实现"画笔式"回放体验。
