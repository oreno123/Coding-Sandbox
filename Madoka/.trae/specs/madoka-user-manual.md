# Madoka 浏览器插件 - 功能总结

## 一、核心功能

### 1. AI 对话功能

**Chat 模式（标准对话）**
- 与 AI 进行自然语言对话
- 支持联网搜索自动判断
- 自动读取当前页面内容作为上下文
- 支持多轮对话和追问



### 2. 搜索功能

- **智能搜索判断**：AI 自动分析用户问题，决定是否需要联网搜索
- **多轮搜索**：支持最多 3 轮关联搜索
- **追问重写**：将依赖上下文的追问转换为独立可搜索的问题
- **搜索引擎**：支持 Bing 和 Google

### 3. 页面内容读取

- **自动读取**：在侧边栏打开时自动读取当前页面内容
- **右键链接总结**：右键点击链接选择"总结此链接"
- **页面引用**：通过 `@` 功能引用当前页面作为对话上下文

### 4. 翻译功能

- **划词翻译**：在网页上选中文本后自动弹出翻译按钮
- **翻译弹窗**：美观的浮动弹窗显示原文和译文
- **问 AI**：将原文发送到侧边栏进行深度问答
- **复制译文**：一键复制翻译结果

### 5. 记忆系统

- **自动保存**：对话结束后自动保存重要信息
- **板块分类**：支持自定义板块（如"工作"、"学习"、"生活"等）
- **记忆权重**：根据使用频率自动调整记忆权重
- **智能清理**：自动清理过期和低价值记忆
- **用户画像**：自动从对话中推断用户偏好和特征
- **记忆管理**：支持固定、删除、查看记忆列表

### 6. Obsidian 同步

- **本地同步**：将记忆同步到本地 Obsidian 笔记库
- **目录选择**：支持选择 Obsidian 库目录
- **手动同步**：一键同步所有未同步的记忆
- **Markdown 格式**：以 Markdown 文件形式保存

## 二、使用方式

### 打开侧边栏

- **点击扩展图标**：点击浏览器工具栏上的 Madoka 图标
- **快捷键**：使用 Chrome 的侧边栏快捷键
- **右键菜单**：右键链接选择"总结此链接"

### 发送消息

1. 在侧边栏底部的输入框中输入问题
2. 按 `Enter` 发送（`Shift+Enter` 换行）
3. 等待 AI 回复

### 使用搜索

- **自动搜索**：AI 自动判断是否需要搜索
- **强制搜索**：输入 `/search 关键词` 强制触发

### 使用翻译

1. 在网页上选中任意文本
2. 点击出现的"翻译"按钮
3. 在弹窗中查看翻译结果
4. 点击"问 AI"将原文发送到侧边栏

### 板块选择

在输入框上方选择板块：
- 让 AI 只关注特定领域的记忆
- 提高对话的针对性和准确性
- 避免无关记忆的干扰

## 三、设置选项

### 模型选择

- `qwen-plus` - 推荐，平衡速度和质量
- `qwen-turbo` - 快速，适合简单问题
- `qwen-max` - 强大，适合复杂任务

### 搜索设置

- 默认搜索引擎：Bing 或 Google
- 最大搜索结果数：3/5/8/10 个

### 记忆设置

- 启用记忆：开启/关闭自动保存
- 用户画像：开启/关闭自动记录偏好
- Obsidian 同步：开启/关闭同步

### 其他配置

- 通义 API Key：输入阿里云通义千问的 API Key
- GitHub Token：可选，用于提高 API 限流
- 暗黑模式：切换浅色/深色主题

## 四、快捷键/快捷操作

### @ 提及功能

输入 `@` 触发上下文引用：
- 标签页
- 书签
- 历史记录
- 当前页面

### 快捷指令

| 指令 | 作用 |
|------|------|
| `/search 关键词` | 强制触发网络搜索 |
| `/clear` | 清空对话 |
| `/help` | 显示帮助 |
| `/read` | 读取当前页面 |
